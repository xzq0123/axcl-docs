query(handle: int, block: bool) → tuple[int, int]#

This API is used to query the status of a called function by using the returned IveHandle of the function. In block mode, the system waits until the function that is being queried is called. In non-block mode, the current status is queried and no action is taken.

Language

Function Prototype

C

AX_S32 AXCL_IVE_Query(AX_IVE_HANDLE IveHandle, AX_BOOL *pbFinish, AX_BOOL bBlock);

python

is_finish, ret = axcl.ive.query(handle, block)
Parameters:

  • handle (int) – IveHandle of a called function. It is entered by users.

  • block (bool) – Flag indicating the block mode or non-block mode.

Returns:

tuple[int, int]

  • is_finish (bool) - Returned status.

  • ret (int) - 0 indicates success, otherwise failure.
