get_stream(chn: int, millisec: int) → tuple[dict, int]#

Get encoded stream of specified channel.

Language

Function Prototype

C

AX_S32 AXCL_VENC_GetStream(VENC_CHN VeChn, AX_VENC_STREAM_T *pstStream, AX_S32 s32MilliSec);

python

stream, ret = axcl.venc.get_stream(chn, millisec)
Parameters:

  • chn (int) – Channel id

  • millisec (int) – Timeout in milliseconds

Returns:

tuple[dict, int]

  • stream (dict) - AX_VENC_STREAM_T

  • ret (int) - 0 indicates success, otherwise failure.

Example

import ctypes

stream, ret = axcl.venc.get_stream(chn, -1)
if ret == 0:
    pack = stream['pack']
    size = pack['len']

    # copy stream data from device to host
    buffer = ctypes.create_string_buffer(size)
    axcl.rt.memcpy(ctypes.addressof(buffer), pack['phy_addr'], size, axcl.AXCL_MEMCPY_DEVICE_TO_HOST)

    # release stream
    axcl.venc.release_stream(chn, stream)
